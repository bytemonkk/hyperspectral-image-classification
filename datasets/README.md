# Dataset Download Links

MF-HSINet is evaluated on multiple benchmark hyperspectral image datasets.

Due to licensing restrictions and repository size limitations, datasets are not included in this repository.

Please download the datasets from the official sources below and place them inside the corresponding folders under `datasets/`.

---

## Indian Pines

Source:

https://www.kaggle.com/datasets/abhijeetgo/indian-pines-hyperspectral-dataset

Folder Structure:

```text
datasets/
└── indian_pines/
```

---
